## Tarea 1.2
### Comandos
Los comandos utilizados
Crear rama
1. git branch v0.2
Moverse a la rama
2. git checkout v0.2
Comprobar que estas en la rama
3. git status
Se crea el archivo
4. git add .
5. git commit -m "Primer commit en la rama"
Configurar rama con el push
6. git push --set-upstream origin v0.2
Moverse a main
7. git checkout main
Juntar ramas
8. git merge v0.2
